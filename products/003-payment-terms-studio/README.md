# Payment Terms Studio (P003)

**Get Paid Toolkit · Portfolio Project 003** — selected by AI-DOS Mission 015 organizational learning.

## Problem

Freelancers lose money when payment expectations are vague. Late chasing (P002) happens because terms were never clear upfront.

## What it does

One form → three copy-ready outputs:

1. **Contract payment clause** — net days, deposit, late fee
2. **Invoice footer** — short line for PDF invoices
3. **Client email** — explain terms when work starts
