# Mission 013: Build the First External Product Using AI-DOS

**Status:** Complete — see [report.md](report.md)  
**Executed by:** Cursor Cloud Agent

## Objective

Use AI-DOS to identify, scope, and build one small useful external product.
This is the first proof that AI-DOS can manage software companies — not just
build more AI-DOS infrastructure.

## Phases

1. **Generate** exactly 5 candidate product ideas (Execution Engine context).
2. **Select** one via scored criteria; reject the other four with reasoning.
3. **Build** a minimal MVP in `products/001-billable-rate-calculator/`.

## Constraints

- No more AI-DOS infrastructure unless absolutely required.
- PHP/Hostinger-friendly; static HTML/CSS/JS acceptable.
- No auth, payments, complex APIs, or fake functionality in v1.
- Product code lives under `products/` — not in compiler, missions (except this folder), or Command Center core.

## Success criteria

- 5 ideas proposed, 1 selected, MVP built and usable.
- Asset Registry updated for product files.
- Mission report complete with deployment path.
